(static) runStudy(scripts, path, path)

Registers analysis scripts and associated listener functions. For each analysis name (identified by object keys), the function expects a script and listener for the result. The analysis script is scheduled to execute in a worker thread during browser idle time. The results from analysis script are forwarded to the listener function.
