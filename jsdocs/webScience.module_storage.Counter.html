Class: Counter

.storage.Counter(counterName)

Class for maintaining persistent counters (e.g., unique IDs).

Constructor

new Counter(counterName)

Create a persistent counter. Note that, because creating a counter requires asynchronous calls (which cannot happen in a constructor), the counter will not be setup until a subsequent call to initialize().

Parameters:
Name Type Description
counterName string

A name that uniquely identifies the counter.
